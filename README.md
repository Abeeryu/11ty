# 11ty – Karateschool John Smit

## Beschrijving

Het project is gebouwd met Eleventy (11ty), een eenvoudig en flexibel static site generator.
Het gebruikt HTML, CSS en JavaScript om een moderne, responsieve en schaalbare website te maken.

### Wanneer gebruik je 11ty?

- Als je een lichte en snelle statische website wilt maken.

- Als je een website wilt met templates, layouts en herbruikbare componenten.

- Als je markdown of HTML content wilt combineren met dynamische templates.

- Als je geen complex JavaScript-framework nodig hebt zoals React of Next.js.

## Installatie

### Clone de repository

git clone https://github.com/Abeeryu/11ty.git

### Ga naar de projectmap

cd 11ty

### Installeer dependencies

npm install

### Start de development server

npm start

### Open de site in je browser

http://localhost:8080

## Technologieën

HTML & CSS – Voor structuur en styling

Vanilla JavaScript – Voor kleine interactieve functies (optioneel)

## Structuur van het project

_11ty/                 – Optionele map voor Eleventy-configuratie
_includes/             – Templates en layouts
_data/                 – Globale data voor templates
index.html             – Hoofdpagina
styles/                – CSS bestanden
package.json           – Dependencies en scripts

11ty genereert automatisch de output in de _site-map (of een andere door jou ingestelde output-map).

## Gebruik

- Voeg pagina’s toe als .html of .njk bestanden in de projectmap.

- Gebruik _includes en layouts voor herbruikbare structuur.

- Styling kan via CSS of Tailwind CSS als je dat toevoegt.

- Start de development server met npm start en de site wordt automatisch ververst bij wijzigingen.
